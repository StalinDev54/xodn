## 🌟 项目简介

**心元 (X Origin Digital Nexus)** 是一个数字生命记忆库项目。我们致力于构建一个可以跨越时空的“数字分身”空间，通过多模态数据整合，为用户保存最珍贵的语言风格、情感偏好与思维逻辑。

> "用代码编织记忆，让每一个平凡的生命光辉都能在数字宇宙中永恒闪耀。"

官方网站:https://xodn.com

### ✨ 核心特性

- 🧠 **多模态展示引擎**：深度融合文字、语音、图像与视频数据，全方位立体化还原生命印记。
- 🔒 **数据主权保护**：基于去中心化存储理念，确保个人记忆资产的永久性与隐私性。
- ♾️ **生命叙事计划**：智能化的记忆碎片整理，自动构建属于个人的生命感官传记。
- 🎨 **极简交互体验**：采用现代化的 UI 设计语言，提供沉浸式的记忆回溯体验。

## 🚀 快速开始

### 🛠️ 环境要求

| 工具                | 最低版本   | 推荐版本   |
| :------------------ | :--------- | :--------- |
| **Node.js**         | `v18.0.0`  | `v20.x`+   |
| **Package Manager** | `pnpm 8.0` | `pnpm 9.x` |

### 📥 安装步骤

1. **克隆仓库**
   ```bash
   git clone <repository-url>
   cd xinyuan-ai-vault
   安装依赖项
   ```

Bash
pnpm install
启动开发环境

Bash
pnpm dev
本地访问：http://localhost:3000

🛠️ 技术栈清单
核心架构
前端: React 19 + TypeScript (严格类型检查)

构建: Vite 6 (极速热更新体验)

路由/状态: React Router + Hooks

视觉与交互
样式: Tailwind CSS 4.0 (实用优先渲染)

动画: Motion 12 (流畅的流体动画效果)

图标: Lucide React (像素级精美图标)

服务端支持
运行环境: Node.js + Express

包管理: pnpm (高效、磁盘占用低)
